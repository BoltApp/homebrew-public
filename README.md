# BoltApp Public

## How do I install these formulae?

`brew install boltapp/public/<formula>`

Or `brew tap boltapp/public` and then `brew install <formula>`.

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
